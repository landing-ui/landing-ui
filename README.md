# landing-ui

> React Components for building landing pages quickly 

[![NPM](https://img.shields.io/npm/v/landing-ui.svg)](https://www.npmjs.com/package/landing-ui) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save landing-ui
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'landing-ui'
import 'landing-ui/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
